# Vim 配置使用教程

本教程针对当前已配置的 Vim（主配置在 `~/.config/vim/vimrc`），帮助你更高效地在项目里浏览与打开文件。

## 1. 基础显示

已启用以下显示设置（无需额外操作）：

- **相对行号 + 绝对行号**：便于跳转与定位
- **当前行高亮**：更清晰定位光标位置
- **状态栏**：显示完整路径、文件类型、编码、文件格式、行列信息

状态栏格式示例：
```
/path/to/file.tsx [tsx] [utf-8] [unix] 120/350:8
```

## 2. 项目目录树（文件树）

使用 Vim 自带 `netrw` 文件树，快捷开关：

- **开关文件树**：`Ctrl + B`

文件树内操作：

- **打开文件**：`Enter`
- **打开文件并自动关闭树**：`o`
- **关闭文件树**：`q`

默认会隐藏大目录：`.git`、`node_modules`、`dist`、`build`、`.next`、`coverage`、`.cache`、`.turbo`

## 3. 快速打开项目内文件或目录

### 3.1 使用命令 `:P`

在 Vim 命令行输入：
```
:P src/subpages/masterPodcast/detail/index
```

说明：

- 如果是 **目录**，会打开该目录的文件树
- 如果是 **文件**，会直接打开
- 如果不写后缀，会自动尝试 `.tsx/.ts/.jsx/.js`
- 若未精确匹配，会在项目根目录下搜索候选并让你选择

### 3.2 使用快捷键 `Ctrl + P`

- 按 `Ctrl + P` 会弹出输入框
- 输入相对路径即可，如：
```
masterPodcast/detail/index
```

如果路径不完整或不确定：
- 会在项目根目录下自动搜索匹配
- 多个候选时提供选择列表

## 4. 搜索功能

- **搜索**：`Ctrl + F` 进入搜索
- **清除高亮**：`Space + h`

搜索体验已优化：
- 输入即搜索（增量搜索）
- 高亮匹配
- 忽略大小写，但若包含大写则区分大小写

## 5. 常用快捷键

| 功能 | 快捷键 |
|------|--------|
| 保存 | `Space + w` |
| 退出 | `Space + q` |
| 保存并退出 | `Space + wq` 或 `Space + x` |
| 退出所有 | `Space + qa` |
| 开关文件树 | `Ctrl + B` |
| 打开路径输入框 | `Ctrl + P` |
| 搜索 | `Ctrl + F` |
| 清除搜索高亮 | `Space + h` |

> 注：`Space` 为 Leader 键。

## 6. 典型使用场景

### 场景 A：快速打开一个文件
```
Ctrl + P
```
输入：
```
masterPodcast/detail/index
```
回车即可打开：
`src/subpages/masterPodcast/detail/index.tsx`

### 场景 B：查看某个目录的结构
```
:P src/subpages/masterPodcast/detail
```
将打开该目录树。

## 7. 常见问题

**Q: `Ctrl + B` 不生效？**  
A: 确保 Vim 在普通模式（按 `Esc`），然后再按 `Ctrl + B`。

**Q: 输入路径找不到？**  
A: 试试只输入部分路径（如 `masterPodcast/detail`），系统会在项目根目录内搜索匹配。
